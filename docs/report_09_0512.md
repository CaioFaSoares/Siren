### System Tray no Linux
- **Problema**: A tentativa de usar a biblioteca `systray` resultou em conflitos de Main Thread (GTK) com o Wails v2, causando `SIGABRT`.

- **Alternativa**: O controle atual é feito via **CLI** ou reabrindo a interface. A implementação de um ícone estável no Linux v2 exigiria comunicação entre processos ou migração para o Wails v3.
